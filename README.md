# Simple ERC-20 Token Factory

This repository provides a streamlined, professional-grade implementation of an ERC-20 token using Solidity and the OpenZeppelin library. It is designed for developers looking to understand the core mechanics of fungible tokens on Ethereum-compatible blockchains.

## Features
* **Standard Compliant:** Fully adheres to the ERC-20 standard.
* **Secure:** Built using audited OpenZeppelin templates.
* **Flat Structure:** All necessary files are located in the root directory for easy navigation.
* **Ready to Deploy:** Includes configuration for Hardhat environments.

## Quick Start
1. Install dependencies: `npm install`
2. Compile contracts: `npx hardhat compile`
3. Run tests: `npx hardhat test`
